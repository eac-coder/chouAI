The Process
How ChouAI Works

Four simple steps from document to insight.

1

Upload Your PDF

Drag and drop or click to select any PDF. ChouAI uses PDF.js to read the file entirely in your browser — nothing is ever uploaded to a server. Your document stays 100% private.

2

Choose Your Output

Select a concise Summary, a deep Explanation of key concepts, or Both. Each mode sends a tailored prompt to the AI for best results.

3

OpenRouter AI Processing

The extracted text is sent to OpenRouter's API, running Mistral 7B Instruct — a powerful open-source language model known for fast, high-quality responses and excellent summarisation abilities.

4

Get Your Results

Results appear instantly on the page. Copy to clipboard or download as a .txt file. No redirects, no emails — everything happens right here.

⚙ Under the Hood

ChouAI uses PDF.js for in-browser PDF text extraction and OpenRouter for AI inference, running mistralai/mistral-7b-instruct. No backend server, no database, no build step — a single HTML file hosted on GitHub Pages, free and private by default.
